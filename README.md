
 
CodeAlpha Student Tracker
Overview
I am happy to work on this new Python program with you. The Student Grade Tracker is a Python-based application designed to manage and track student information efficiently. This tool aims to simplify organizing student and their grades and averages their grades for their overall school performance.

Features
Add, update, and delete student records.
Academic performance.
Generate reports for students and classes.
User-friendly interface with streamlined workflows.

Installation:
Clone this repository:
bash
git clone https://github.com/blovelace84/CodeAlpha_Student_Tracker.git
Navigate to the project directory:
bash
cd CodeAlpha_Student_Tracker
Install the required dependencies:
bash
cd grade_tracker
python main.py
Follow the on-screen instructions to interact with the system.
